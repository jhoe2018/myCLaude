# 进制运算与转换

一个通用的进制转换与运算工具，支持任意进制(2-36)的互转和四则运算。

代码实现: `${CLAUDE_SKILL_DIR}/base_convert.py`

## 输入格式

1. **纯转换**: `数值 [源进制] [目标进制]`
   - `FF 16 2` — 十六进制FF → 二进制
   - `1010 2 10` — 二进制1010 → 十进制
   - `255 10 16` — 十进制255 → 十六进制
   - 省略源进制默认十进制，省略目标进制默认打印所有常用进制

2. **运算表达式**: `数值1 运算符 数值2 [进制]`
   - `FF + 1 16` — 十六进制 FF + 1
   - `1010 + 111 2` — 二进制加法
   - `1A * 3 16` — 十六进制乘法

3. **简短查询**:
   - `0xFF` — 自动识别为十六进制
   - `0b1010` — 自动识别为二进制
   - `0o77` — 自动识别为八进制

## 操作步骤

1. 解析用户输入，识别数值、运算符、进制
2. 调用 `python ${CLAUDE_SKILL_DIR}/base_convert.py <表达式>` 执行计算
3. 将结果呈现给用户

## 注意事项

- 进制范围: 2-36（使用 0-9 和 A-Z 表示）
- 二进制转八进制: 从右往左每3位一组
- 二进制转十六进制: 从右往左每4位一组
- 小数转换: 整数部分用除法取余，小数部分用乘法取整
